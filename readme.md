## üèÅ Starting

```bash
# Clone this project
$ git clone https://github.com/timoteostifft/paypal_node.git
# Access
$ cd ./paypal_node
```

### Node

```bash
# Install dependencies
$ npm install
# Run the project
$ npm run dev
# The server will initialize at <http://localhost:3000>
```

### Docker

```bash
# Install project
$ docker-compose build
# Run project
$ docker-compose up
# The server will initialize at <http://localhost:3000>
```
